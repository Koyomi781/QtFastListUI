# QtFastListUI
It can creat a concise Qlistwidget interface, Including the sidebar

# 介绍/Introduction
**感谢浏览本项目**
*该项目是基于pyqt5，通过重写QListWidgets,QPushButton类，实现快捷创建一个简洁的折叠窗口
*其中包括简洁美观的侧边栏，让你更快的把很多可以折叠起来的窗口整合到一个页面上，并把这个页面
绑定到左侧的侧边栏上
*欢迎大家使用本项目整合制作一些工具
